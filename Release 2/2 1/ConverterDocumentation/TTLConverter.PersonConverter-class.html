Class PersonConverter

source code

This Class contains a TTLSet Object with semantic Data. It generates this Data by evaluating a dictinary. This dictionary have to be put in by contructing the class.

source code 

Generic Method to generate new TTLRessource with Relationship to the TTLRessource Object of the person.

Parameters:
  • predicate - The property as URI which describes the Connection to the new TTLRessource object
  • type - The type of the new TTLRessource object
  • label - the label of the new TTLRessource object
  • nr - A Numer to create a unique URI for the new TTLRessource if needed (more then one Ressource with identical type and label)
Returns:
the new TTLRessource object

generateRelToGivenRes(self, subjectTTLRes, predicate, type, newLabel, notInternnameLink=True, unique=False)

source code 

Add new TTLRessource object and integrates relation to subjectTTLRessource There is no connction to the persons TTLRessource Object

Parameters:
  • subjectTTLRes - The existing TTLRessource to whom the new relation should be integrated
  • predicate - The property as URI which describes the Connection to the new TTLRessource object
  • type - The type of the new TTLRessource object
  • newLabel - the label of the new TTLRessource object if it is a set (label, countryCode) the countrycode will be added to the Label
Returns:
the new TTLRessource object
